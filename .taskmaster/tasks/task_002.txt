# Title: Setup CSS Keyframes for Marquee Animation
# Status: pending
# Dependencies: None
# Priority: high
# Description: Add @keyframes marquee animation to globals.css for tech stack scrolling
# Details:
Update `src/app/globals.css` to add marquee keyframes:

**Add to bottom of file:**
```css
@keyframes marquee {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(-50%);
  }
}
```

**Also add Tailwind layer for custom animation:**
```css
@layer utilities {
  .animate-marquee {
    animation: marquee 30s linear infinite;
  }
}
```

**Purpose:**
- Creates horizontal scrolling effect for tech logos
- Smooth infinite loop animation
- NO Framer Motion - pure CSS

**Note:** Content will be duplicated in component to create seamless loop
