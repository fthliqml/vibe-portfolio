- pageVariants: hidden, visible, exit with proper transitions

Best practices from motion.instruction.md:
- Always use "use client" directive
- Define variants outside component
- Use TypeScript for props (ReactNode for children)
- Avoid useEffect for animations
- Use simple duration-based transitions (no string easing names)
- Proper AnimatePresence with unique key props
